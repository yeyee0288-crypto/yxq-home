# Excel 数据互通工具

这是一个 Python 桌面应用，用来在两个 Excel 文件之间进行数据互通和同步。

## 功能特性

✅ **自动列匹配** - 自动识别相同或相似的列名  
✅ **灵活映射** - 支持自定义列映射关系  
✅ **数据同步** - 一键将源文件数据复制到目标文件  
✅ **简单易用** - 直观的图形化界面  
✅ **安全保存** - 支持另存为新文件，不覆盖原文件  

## 安装

### 1. 安装 Python（3.7+）
- 下载地址：https://www.python.org/downloads/

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

或者手动安装：

```bash
pip install openpyxl pandas
```

## 使用方法

### 启动应用

```bash
python excel_sync_tool.py
```

### 操作步骤

1. **选择源文件** - 点击"浏览"选择第一个 Excel 文件 (A)
2. **选择目标文件** - 点击"浏览"选择第二个 Excel 文件 (B)
3. **加载文件** - 点击"加载文件"按钮
4. **自动匹配** - 点击"自动匹配 →"自动识别相似列名
5. **查看映射** - 点击"查看映射"查看匹配结果
6. **手动调整**（可选）- 如果自动匹配不满意，可以手动设置映射关系
7. **执行同步** - 点击"执行同步"开始数据同步
8. **保存文件** - 点击"保存文件"保存同步后的数据

## 工作原理

### 列匹配算法

应用使用相似度算法来自动匹配列名：
- 计算源文件列名与目标文件列名的相似度
- 相似度 > 60% 时认为是匹配的列
- 选择最相似的列进行映射

### 数据同步

- 源文件的数据会被复制到目标文件的对应列
- 只填充目标文件已有的行数范围
- 原始数据不会被覆盖，而是复制到新位置

## 常见问题

### Q: 支持哪些 Excel 格式？
A: 支持 `.xlsx` 和 `.xls` 格式

### Q: 会覆盖原文件吗？
A: 不会。应用会要求你选择新的保存位置，原文件不会被修改

### Q: 支持多个 Sheet 吗？
A: 当前版本只支持第一个 Sheet。如需扩展，可以修改代码

### Q: 自动匹配不准确怎么办？
A: 可以点击"清除映射"后手动设置列映射关系

## 进阶用法

### 修改相似度阈值

在 `excel_sync_tool.py` 中找到 `auto_match` 方法，修改这行：

```python
if score > best_score and score > 0.6:  # 相似度阈值为 0.6
```

- 提高阈值（如 0.8）：只匹配高度相似的列
- 降低阈值（如 0.4）：匹配更多不太相似的列
